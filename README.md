# Daily Diet API

The project is a diet API where an authenticated user can log their meals, marking whether they are within their diet or not. With this the user can obtain his metrics and control his health. 

The possible options that the user can have, are registering and searching for his metrics, and registering, updating, removing and searching for his meals.

The project was made using `Fastify` 

To start the project you need:

``` 
-> .env

NODE_ENV=development
DATABASE_URL="file:./db/dev.db"
```

 ```
 npm install
 ```
 ```
 npm run dev
 ```
